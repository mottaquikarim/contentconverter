# Content Converter

Converts Eggers content into RevealJS Markdown.

**[Example Output](https://github.com/mottaquikarim/FEWDRemote)**


## Usage

```
$ git clone [this_repo]
$ cd [this_repo]
$ virtualenv -p python3 .ven
$ source .venv/bin/activate
$ pip install -r requirements.txt
$ python script.py
```
